FAQ

Is Mu free to use?
Yes! Create an account and start using Mu immediately at no cost.

Can I self-host Mu?
Absolutely. Mu is open source and runs as a single Go binary. Check the GitHub repo for installation instructions.

What about membership?
Optional membership supports ongoing development and hosting. Members get early access to new features and a voice in the project's direction. The platform remains free for everyone.

How is this different from big tech platforms?
No ads, no algorithmic feeds, no data mining. Just simple, useful tools that work for you.
